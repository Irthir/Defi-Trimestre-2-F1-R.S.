1] SDL ou SDL2 : Donnez les différences.

SDL est la version précédente de SDL2 est n'est plus mise à jour.
SDL2 est donc la version actuelle pour la SDL.

2]Mise en Oeuvre SDL Image, pourquoi SDL_Image ?
SDL_Image permet de créer des textures SDL2 à partir d'images de type : ICO, CUR, BMP, GIF, JPG, LBM, PCX, PNG, PNM, TIF, XCF, XPM, XV, WEBP.
Ce qui permet de créer des textures sans avoir à se limiter aux BitMaps.
